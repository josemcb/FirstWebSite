GitHub

Es un sitio web y un servicio en la nube que ayuda a los desarrolladores a almacenar y administrar su código, al igual que llevar un registro y control de cualquier cambio sobre este código.

HTML

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto

CSS

Es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML. CSS separa el contenido de la representación visual del sitio.

JavaScript

Es el único lenguaje de programación que funciona en los navegadores de forma nativa (lenguaje interpretado sin necesidad de compilación). Por tanto se utiliza como complemento de HTML y CSS para crear páginas webs.

DomModel

El Modelo de Objetos del Documento (DOM) es una interfaz de programación de aplicaciones (API) para documentos validos HTML y bien construidos XML. Define la estructura lógica de los documentos y el modo en que se accede y manipula.

VisualCode

Visual Studio Code es un editor de código redefinido y optimizado para construir y depurar aplicaciones web y en la nube modernas. Visual Studio Code es gratuito y está disponible en tu plataforma favorita: Linux, macOS y Windows.

Footer

Rrepresenta un pie de página para su contenido de sección más cercano o elemento raíz de sección. Suele contener información sobre el autor de la sección, datos de copyright o enlaces a documentos relacionados.

Head

Contiene información legible por la máquina (metadatos) sobre el documento, como su título, scripts y hojas de estilo.

Body

Representa el contenido de un documento HTML. Sólo puede haber un elemento body en un documento.

Title

Define el título del documento que se muestra en la barra de título de un navegador o en la pestaña de una página. Sólo contiene texto; las etiquetas dentro del elemento se ignoran.

Link

especifica las relaciones entre el documento actual y un recurso externo. Este elemento se utiliza más comúnmente para enlazar a hojas de estilo, pero también se utiliza para establecer iconos del sitio entre otras cosas.

Style

Contiene información de estilo para un documento, o parte de un documento. Contiene CSS, que se aplica al contenido del documento que contiene el elemento style.

Button

Representa un botón sobre el que se puede hacer clic, que se utiliza para enviar formularios o en cualquier lugar de un documento para una funcionalidad de botón accesible y estándar.

Href

La URL a la que apunta el hipervínculo. Los enlaces no están restringidos a las URLs basadas en HTTP - pueden utilizar cualquier esquema de URL soportado por los navegadores.

Meta

Representa metadatos que no pueden ser representados por otros elementos meta de HTML, como base, enlace, script, estilo o título.

Class

El atributo global class es una lista separada por espacios de las clases del elemento que distingue entre mayúsculas y minúsculas. Las clases permiten a CSS y Javascript seleccionar y acceder a elementos específicos a través de los selectores de clase.
